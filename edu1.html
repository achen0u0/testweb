系統弱點掃描：

針對主機作業系統、網路設備進行安全問題檢測，且依據不同設備、網段和重要性，設定合適的掃描政策。


檢測方式： （１） 定期遠端或到場掃描檢測
（２） 採用Plugin和CEV進行弱點檢測
檢測報告： （１）檢測時間、檢測方式、檢測工具說明
（２）弱點摘要和分析說明
（３）安全強化建議
後續服務： （１）提供複測

網頁弱點掃描：

本團隊依客戶提供的網站進行檢測，檢測功能有整體架構檢測、 Web Application安全檢測（如：SQL Injection、Cross-site scripting、Remote File Inclusion弱點項目檢測）、 CGI檢測（如：POST、GET）。


檢測方式： （１）定期遠端或到場掃描檢測
（２）網站目錄進行全面性的網頁攀爬、URL檢測
檢測報告： （１）檢測時間、檢測方式、檢測工具說明
（２）弱點摘要和分析說明
（３）安全強化建議
後續服務： （１）提供複測
